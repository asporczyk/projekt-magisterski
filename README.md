# Projekt magisterski / Master's project

## Project Setup

### Install packages
```sh
npm install
```

### Run application for Development

```sh
npm run dev
```

### Build application for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Run End-to-End Test with [Cypress](https://www.cypress.io/) (UI)
```shell
npm run cypress
```
